# 💰 Invicta Finanças

Dashboard financeiro pessoal com foco em usabilidade e design intuitivo do público jovem ao mais experiente.

##Link de acesso: invicta-financas-production.up.railway.app

## 🚀 Funcionalidades

- **Autenticação** — cadastro e login com senha criptografada (bcrypt)
- **Dashboard** — saldo, receita prevista, limite de gastos e gráficos em tempo real
- **Transações** — cadastro, edição e exclusão de receitas e despesas por categoria
- **Metas financeiras** — acompanhamento de progresso com barra visual
- **Relatórios** — filtro por mês, gráficos de pizza e linha com dados reais
- **Exportação** — relatórios em PDF e Excel
- **Notificações** — alertas quando os gastos se aproximam ou ultrapassam o limite
- **Dark mode** — alternância entre tema claro e escuro
- **Acessibilidade** — controle de tamanho de fonte

## 🛠️ Tecnologias

- **Front-End:** HTML5, CSS3, JavaScript, TailwindCSS, Chart.js, Feather Icons
- **Back-End:** PHP 8.3 com PDO
- **Banco de Dados:** MySQL
- **Versionamento:** Git & GitHub
